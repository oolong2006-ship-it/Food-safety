# 🛡️ نظام سلامة الغذاء و GMP — منصة إدارة الجودة والتفتيش

تطبيق ويب احترافي لفِرق الجودة وسلامة الغذاء وممارسات التصنيع الجيدة (GMP) في **المطاعم والكافيهات والمصانع الغذائية** الخاضعة للتفتيش والرقابة (هيئة الغذاء والدواء / البلديات).

يعمل التطبيق **بالكامل دون اتصال بالإنترنت**، وتُحفَظ جميع البيانات محليًا على جهاز المستخدم (Local Storage) — لا حاجة لخادم أو قاعدة بيانات.

---

## ✨ الوحدات الرئيسية

| الوحدة | الوصف |
|---|---|
| 📊 **لوحة المعلومات** | مؤشرات أداء فورية: جاهزية التفتيش، نسبة الامتثال، التجاوزات، التنبيهات |
| 📋 **التفتيش الذاتي و GMP** | قوائم تدقيق جاهزة (GMP / الفحص اليومي) مع احتساب النتيجة وإنشاء مخالفات تلقائيًا |
| 🌡️ **مراقبة درجات الحرارة** | تسجيل حرارة الثلاجات/المجمدات/الحفظ الساخن/الطهي مع كشف التجاوزات تلقائيًا |
| 👥 **العاملون والشهادات الصحية** | متابعة الكوادر وصلاحية الشهادات الصحية والتدريب مع تنبيهات الانتهاء |
| ⚠️ **عدم المطابقة (CAPA)** | توثيق المخالفات، تحليل الأسباب الجذرية، ومتابعة الإجراءات التصحيحية حتى الإغلاق |
| 🚚 **اعتماد الموردين** | تقييم الموردين ومتابعة صلاحية تراخيصهم |
| 🧹 **التنظيف ومكافحة الآفات** | جدول التنظيف والتعقيم وسجل زيارات مكافحة الآفات |
| 📷 **الرصد بالتصوير الذكي** | تصوير الموقع ورصد المخالفات تلقائيًا بالذكاء الاصطناعي مع توليد الإجراء التصحيحي والوقائي |
| 📚 **المواصفات والمعايير** | قاعدة معرفة سعودية (SFDA) وخليجية (GSO) وعالمية (Codex / ISO 22000 / HACCP) |
| 📈 **التقارير والجاهزية** | تقرير شامل وقائمة تحقق قبل التفتيش قابلة للطباعة |

## 🤖 ميزات الذكاء الاصطناعي (Claude)

يدمج النظام نموذج **Claude (`claude-opus-4-8`)** لإضافة قدرات ذكية:

- **رصد المخالفات من الصور تلقائيًا** — التقط صورة لموقع العمل، فيحلّلها النموذج (دعم الرؤية) ويرصد مخالفات سلامة الغذاء وGMP.
- **توليد الإجراء التصحيحي والوقائي** — لكل مخالفة مرصودة يقترح إجراءً تصحيحيًا فوريًا وإجراءً وقائيًا لمنع التكرار، مع الإشارة للمواصفة المرجعية، وزر لإنشاء حالة عدم مطابقة جاهزة.
- **تقييم بنود التفتيش وتحديثها** — يقيّم وضوح كل بند ومستوى خطورته، ويقترح صياغة محسّنة قابلة للقياس مع إمكانية تطبيقها بنقرة.
- **مدعوم بالمواصفات** — تُحقَن المواصفات السعودية والخليجية والعالمية وحدودها الحرجة في سياق النموذج لضمان توافق المخرجات معها.

### التفعيل
من **الإعدادات → الذكاء الاصطناعي**: أدخل مفتاح Claude API ثم فعّل الخدمة. يُحفظ المفتاح محليًا على جهازك فقط، وتُرسل الطلبات مباشرة إلى خدمة Claude.

> **وضع بدون مفتاح:** تعمل الميزات بمحرك احتياطي محلي مبني على قاعدة المواصفات (يقترح الإجراءات من الملاحظة النصية)، أما الرصد الآلي من الصور فيتطلب تفعيل الخدمة.

## 🚀 التشغيل

افتح `index.html` مباشرةً في المتصفح، أو شغّل خادمًا محليًا:

```bash
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

اختر دورًا للدخول (مدير الجودة / مشرف سلامة الغذاء / مفتش)، ويبدأ النظام ببيانات تجريبية واقعية.

## 🧱 البنية التقنية

- HTML5 + CSS3 (تصميم RTL عربي متجاوب) + JavaScript خالص — بدون أي مكتبات خارجية.
- `assets/js/store.js` — طبقة البيانات والتخزين والمؤشرات المحسوبة.
- `assets/js/standards.js` — قاعدة المواصفات (سعودية/خليجية/عالمية) وخرائط الإجراءات.
- `assets/js/ai.js` — طبقة الذكاء الاصطناعي (Claude API عبر `fetch` + محرك احتياطي محلي).
- `assets/js/ui.js` — مكونات الواجهة (نوافذ، تنبيهات، شارات).
- `assets/js/views.js` — شاشات الوحدات الوظيفية.
- `assets/js/app.js` — التنقل والجلسة والتهيئة.

## 🔒 الخصوصية

جميع البيانات محلية على الجهاز. يمكن **تصدير/استيراد** نسخة احتياطية بصيغة JSON من الشريط العلوي أو من الإعدادات.
